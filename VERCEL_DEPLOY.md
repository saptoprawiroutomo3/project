# Deploy to Vercel Button

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fsaptoprawiroutomo%2FPengadepan&project-name=inter-media-app&repository-name=inter-media-app&root-directory=inter-media-app&env=MONGODB_URI,NEXTAUTH_SECRET,NEXTAUTH_URL&envDescription=Environment%20variables%20needed%20for%20the%20application&envLink=https%3A%2F%2Fgithub.com%2Fsaptoprawiroutomo%2FPengadepan%2Fblob%2Fmain%2Finter-media-app%2F.env.production.template)

## Quick Deploy Steps:
1. Click the button above
2. Connect your GitHub account
3. Set environment variables:
   - MONGODB_URI: Use MongoDB Atlas free tier
   - NEXTAUTH_SECRET: your-secret-key-here-2026-inter-media-app
   - NEXTAUTH_URL: https://your-app.vercel.app
4. Deploy automatically

## MongoDB Atlas (Free):
1. Go to https://cloud.mongodb.com
2. Create free cluster
3. Get connection string
4. Use in MONGODB_URI
